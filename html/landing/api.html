REST API v1

Tài liệu API

Tích hợp WemakeUI Cloud vào hệ thống của bạn. Tất cả endpoint trả về JSON, xác thực qua Bearer token.

Authentication
  • POST /auth/token
  • POST /auth/revoke
Users
  • GET /users
  • GET /users/:id
Projects
  • POST /projects
Webhooks
  • POST /webhooks
POST/v1/auth/token

Lấy access token từ API key và secret. Token có hiệu lực 1 giờ.

Tham số (body)
TênKiểuBắt buộcMô tả
client_idstringCóAPI key của ứng dụng
client_secretstringCóSecret tương ứng
scopestringKhôngPhạm vi quyền, mặc định read
Phản hồi 
{
  "access_token": "wmk_live_a1b2c3d4e5f6g7h8",
  "token_type": "Bearer",
  "expires_in": 3600,
  "scope": "read"
}
GET/v1/users

Liệt kê người dùng trong workspace, hỗ trợ phân trang.

Tham số (query)
TênKiểuMặc địnhMô tả
pageinteger1Số trang
per_pageinteger25Số bản ghi/trang (tối đa 100)
rolestring—Lọc theo vai trò: admin, member
Phản hồi 
{
  "data": [
    { "id": "usr_01", "name": "Nguyễn Văn A", "email": "a@congty.vn", "role": "admin" },
    { "id": "usr_02", "name": "Trần Thị B", "email": "b@congty.vn", "role": "member" }
  ],
  "page": 1,
  "per_page": 25,
  "total": 42
}
POST/v1/projects

Tạo một project mới trong workspace hiện tại.

Tham số (body)
TênKiểuBắt buộcMô tả
namestringCóTên project
descriptionstringKhôngMô tả ngắn
visibilitystringKhôngprivate (mặc định) hoặc internal
Yêu cầu 
POST /v1/projects
Authorization: Bearer wmk_live_...
Content-Type: application/json

{ "name": "Dự án WebApp 2026", "visibility": "internal" }
Phản hồi 
{
  "id": "prj_8f3a",
  "name": "Dự án WebApp 2026",
  "visibility": "internal",
  "created_at": "2026-05-12T09:30:00+07:00"
}
POST/v1/webhooks

Đăng ký webhook để nhận thông báo sự kiện qua HTTP POST.

Tham số (body)
TênKiểuBắt buộcMô tả
urlstringCóEndpoint nhận sự kiện (HTTPS)
eventsarrayCóDanh sách sự kiện, ví dụ ["project.created","user.invited"]
secretstringKhôngDùng để ký HMAC payload
Phản hồi 
{
  "id": "whk_2c91",
  "url": "https://app.congty.vn/hooks/wemakeui",
  "events": ["project.created", "user.invited"],
  "status": "active"
}
